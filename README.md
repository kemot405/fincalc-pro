# FinCalc Pro - Starter (TypeScript + Next.js + Tailwind)

## Quick start

1. Install dependencies
```
npm install
```

2. Run dev server
```
npm run dev
```

3. Open http://localhost:3000

This starter contains:
- homepage in `app/page.tsx`
- placeholder pages: `/kalkulatory`, `/centrum-wiedzy`, `/kontakt`, `/logowanie`
- Tailwind config and basic styles in `styles/globals.css`

Logo is in `public/logo.svg`.
